ক্রয় প্রতিবেদন (Purchase Report)

সরবরাহকারী খতিয়ান (Supplier Ledger)

প্রতিটি সরবরাহকারীর সম্পূর্ণ হিসাব খতিয়ান দেখুন। ডেবিট-ক্রেডিট সব লেনদেনের বিবরণ এখানে পাবেন। কোন সরবরাহকারীর কত টাকা বকেয়া আছে তা সহজেই বুঝতে পারবেন।

  1. সাইডবার থেকে MIS Report → Purchase Report → Supplier Ledger ক্লিক করুন
  2. সরবরাহকারীর নাম ড্রপডাউন থেকে নির্বাচন করুন অথবা সব দেখুন
  3. তারিখ সীমা নির্ধারণ করুন (From Date - To Date)
  4. খতিয়ানে ডেবিট (ক্রয়), ক্রেডিট (পরিশোধ) ও ব্যালেন্স দেখুন
  5. PDF বা CSV তে রপ্তানি করতে পারবেন

দৈনিক ক্রয় প্রতিবেদন (Daily Purchase Report)

প্রতিদিনের ক্রয়ের সামগ্রিক প্রতিবেদন দেখুন। কোন তারিখে কত টাকার প্রোডাক্ট কেনা হলো তার বিবরণ পাবেন। দৈনিক ক্রয় পরিমাণ তুলনা করে ব্যবসার ক্রয় প্রবণতা বুঝতে সাহায্য করে।

  1. MIS Report → Purchase Report → Daily Purchase Report ক্লিক করুন
  2. তারিখ নির্বাচন করুন যে দিনের রিপোর্ট দেখতে চান
  3. সেই দিনের সকল ক্রয় অর্ডারের তালিকা দেখুন
  4. মোট ক্রয়ের পরিমাণ নিচে সামগ্রিক হিসাবে দেখাবে

সরবরাহকারীভিত্তিক ক্রয় (Suppliers Wise Purchase)

কোন সরবরাহকারী থেকে কত টাকার প্রোডাক্ট কেনা হয়েছে তার তুলনামূলক প্রতিবেদন। বেশি কেনা সরবরাহকারী শনাক্ত করতে এবং ক্রয় কৌশল নির্ধারণে সাহায্য করে।

  1. MIS Report → Purchase Report → Suppliers Wise Purchase ক্লিক করুন
  2. তারিখ সীমা নির্ধারণ করুন
  3. প্রতিটি সরবরাহকারীর মোট ক্রয়ের পরিমাণ দেখুন
  4. PDF/CSV রপ্তানি করুন

সরবরাহকারীকে নগদ প্রদান (Supplier Cash Delivery)

সরবরাহকারীদের কাছে কত নগদ অর্থ প্রদান করা হয়েছে তার রেকর্ড। পেমেন্ট হিস্ট্রি ট্র্যাকিং ও নগদ প্রবাহ বিশ্লেষণে সাহায্য করে।

  1. MIS Report → Purchase Report → Supplier Cash Delivery ক্লিক করুন
  2. তারিখ সীমা ও সরবরাহকারী নির্বাচন করুন
  3. সকল নগদ প্রদানের বিবরণ দেখুন

সরবরাহকারীর বকেয়া প্রতিবেদন (Suppliers Due Report)

কোন সরবরাহকারীর কত টাকা বকেয়া আছে তার সম্পূর্ণ প্রতিবেদন। বকেয়া পরিশোধের পরিকল্পনা করতে এবং ক্রেডিট সম্পর্ক পরিচালনায় সাহায্য করে।

  1. MIS Report → Purchase Report → Suppliers Due Report ক্লিক করুন
  2. সরবরাহকারীভিত্তিক বকেয়া তালিকা দেখুন
  3. মোট বকেয়া পরিমাণ নিচে দেখাবে
  4. PDF রপ্তানি করে রাখুন
নিয়মিত এই রিপোর্ট দেখুন যাতে কোনো সরবরাহকারীর বকেয়া অতিরিক্ত না হয়। বকেয়া বাড়লে সরবরাহ বিঘ্ন হতে পারে।

মডেলভিত্তিক ক্রয় (Model Wise Purchase)

কোন মডেলের প্রোডাক্ট কত পরিমাণে কেনা হয়েছে তার প্রতিবেদন। চাহিদা অনুযায়ী ক্রয় পরিকল্পনা করতে সাহায্য করে।

  1. MIS Report → Purchase Report → Model Wise Purchase ক্লিক করুন
  2. ক্যাটাগরি বা মডেল অনুযায়ী ফিল্টার করুন
  3. প্রতিটি মডেলের ক্রয় পরিমাণ ও মোট খরচ দেখুন

ভ্যাট প্রতিবেদন (Vat Report)

সকল ক্রয়ে ভ্যাটের সামগ্রিক হিসাব। সরকারি ভ্যাট রিটার্ন জমা দিতে এবং কর পরিকল্পনায় সাহায্য করে।

  1. MIS Report → Purchase Report → Vat Report ক্লিক করুন
  2. তারিখ সীমা নির্ধারণ করুন
  3. প্রতিটি ক্রয়ে ভ্যাটের পরিমাণ দেখুন
  4. মোট ভ্যাটের সামগ্রিক হিসাব নিচে দেখাবে
বিক্রয় প্রতিবেদন (Sales Report)

দৈনিক বিক্রয় প্রতিবেদন (Daily Sales Report)

প্রতিদিনের বিক্রয়ের সম্পূর্ণ প্রতিবেদন। কোন তারিখে কত টাকার প্রোডাক্ট বিক্রি হলো, নগদ ও ক্রেডিট বিক্রয়ের বিভাজন সহ বিস্তারিত তথ্য পাবেন। দৈনিক আয় ট্র্যাকিংয়ের জন্য অপরিহার্য।

  1. MIS Report → Sales Report → Daily Sales Report ক্লিক করুন
  2. তারিখ নির্বাচন করুন
  3. সেই দিনের সকল বিক্রয় অর্ডারের তালিকা দেখুন
  4. নগদ ও ক্রেডিট বিক্রয়ের পরিমাণ আলাদাভাবে দেখুন
  5. মোট বিক্রয়ের সামগ্রিক হিসাব নিচে দেখাবে
প্রতিদিন শেষে এই রিপোর্ট দেখে দিনের আয়ের সারসংক্ষেপ নিন। সপ্তাহ বা মাসের তুলনা করতে তারিখ পরিবর্তন করুন।

প্রতিস্থাপন প্রতিবেদন (Replacement Report)

কতগুলো প্রোডাক্ট প্রতিস্থাপন করা হয়েছে তার প্রতিবেদন। প্রোডাক্টের মান নিয়ন্ত্রণ ও সরবরাহকারীর সাথে আলোচনার জন্য গুরুত্বপূর্ণ।

  1. MIS Report → Sales Report → Replacement Report ক্লিক করুন
  2. তারিখ সীমা নির্ধারণ করুন
  3. প্রতিস্থাপিত প্রোডাক্টের বিবরণ দেখুন

মডেলভিত্তিক বিক্রয় (Model Wise Sales)

কোন মডেলের প্রোডাক্ট সবচেয়ে বেশি বিক্রি হচ্ছে তার প্রতিবেদন। বেস্ট সেলার প্রোডাক্ট চিহ্নিত করে স্টক ও ক্রয় পরিকল্পনা নির্ধারণে সাহায্য করে। কোন প্রোডাক্ট বেশি বিক্রি হয় তা চিনতে এই রিপোর্ট অপরিহার্য।

  1. MIS Report → Sales Report → Model Wise Sales ক্লিক করুন
  2. ক্যাটাগরি বা ব্র্যান্ড অনুযায়ী ফিল্টার করুন
  3. প্রতিটি মডেলের বিক্রয় সংখ্যা ও আয় দেখুন
  4. সবচেয়ে বেশি বিক্রি হওয়া মডেলগুলো শীর্ষে থাকবে
বেশি বিক্রি হওয়া প্রোডাক্টগুলোর স্টক যাতে কম না হয় সেদিকে খেয়াল রাখুন। Model Wise Sales রিপোর্ট দেখে Auto PO ব্যবহার করে স্বয়ংক্রিয়ভাবে ক্রয় অর্ডার তৈরি করুন।
কিস্তি বিক্রয় প্রতিবেদন (Hire Sales Report)

কিস্তি আদায় (Installment Collection)

কিস্তিতে বিক্রি করা প্রোডাক্টের মাসিক কিস্তি আদায়ের রেকর্ড। কোন গ্রাহক কত টাকা কিস্তি দিয়েছে এবং কত বকেয়া আছে তার সম্পূর্ণ বিবরণ। কিস্তি বিক্রয়ের নগদ প্রবাহ পরিচালনার জন্য এই পেজ অত্যন্ত গুরুত্বপূর্ণ।

  1. MIS Report → Hire Sales Report → Installment Collection ক্লিক করুন
  2. গ্রাহক নাম বা তারিখ দিয়ে ফিল্টার করুন
  3. প্রতিটি কিস্তি পেমেন্টের বিবরণ দেখুন - তারিখ, পরিমাণ, গ্রাহক
  4. মোট আদায়কৃত কিস্তির সামগ্রিক হিসাব নিচে দেখাবে
  5. রপ্তানি করতে PDF/CSV আইকনে ক্লিক করুন
প্রতি মাসের শুরুতে আসন্ন কিস্তির তালিকা দেখে আদায় পরিকল্পনা করুন। নিয়মিত কিস্তি আদায় না হলে খেলাপি গ্রাহক রিপোর্ট দেখুন।

আসন্ন কিস্তি (Upcoming Installment)

আগামী কোন তারিখে কোন গ্রাহকের কিস্তি পেমেন্ট নির্ধারিত আছে তার পূর্বাভাস। আদায়কর্মীদের পরিকল্পনা করতে এবং গ্রাহকদের রিমাইন্ডার পাঠাতে সাহায্য করে।

  1. MIS Report → Hire Sales Report → Upcoming Installment ক্লিক করুন
  2. তারিখ সীমা নির্ধারণ করুন (যেমন: আগামী ৩০ দিন)
  3. আসন্ন সকল কিস্তি পেমেন্টের তালিকা দেখুন
  4. গ্রাহকদের SMS পাঠাতে Send SMS পেজ ব্যবহার করুন

খেলাপি গ্রাহক (Defaulting Customer)

যেসব গ্রাহক কিস্তি সময়মতো পরিশোধ করেনি তাদের বিস্তারিত তালিকা। কত কিস্তি বকেয়া, কত দিনের বকেয়া তা দেখতে পারবেন। ঋণ আদায় ও আইনি ব্যবস্থা নিতে সাহায্য করে।

  1. MIS Report → Hire Sales Report → Defaulting Customer ক্লিক করুন
  2. সকল খেলাপি গ্রাহকের তালিকা দেখুন
  3. বকেয়ার পরিমাণ ও দিন সংখ্যা অনুযায়ী সাজান
  4. দীর্ঘমেয়াদী খেলাপিদের আলাদাভাবে চিহ্নিত করুন
খেলাপি গ্রাহকের তালিকা গোপনীয় - শুধুমাত্র অ্যাডমিন ও ম্যানেজার দেখতে পারবেন। তালিকা বিতরণ করবেন না।

খেলাপি গ্রাহক সারসংক্ষেপ (Default Customer Summary)

সকল খেলাপি গ্রাহকের সারসংক্ষেপ রিপোর্ট। মোট খেলাপি পরিমাণ, গ্রাহক সংখ্যা ও গড় বকেয়া দেখতে পারবেন।

  1. MIS Report → Hire Sales Report → Default Customer Summary ক্লিক করুন
  2. সারসংক্ষেপ টেবিলে মোট খেলাপি পরিমাণ দেখুন
  3. ক্যাটাগরি বা সময়কাল অনুযায়ী বিশ্লেষণ করুন

কিস্তি হিসাব বিবরণ (Hire Account Details)

প্রতিটি কিস্তি বিক্রয়ের সম্পূর্ণ হিসাব বিবরণ - মোট মূল্য, ডাউন পেমেন্ট, পরিশোধিত কিস্তি, বকেয়া কিস্তি, সুদ ইত্যাদি।

  1. MIS Report → Hire Sales Report → Hire Account Details ক্লিক করুন
  2. নির্দিষ্ট হায়ার সেলস নির্বাচন করুন
  3. সম্পূর্ণ পেমেন্ট শিডিউল ও বর্তমান অবস্থা দেখুন
  4. সুদ ও মূলধনের বিভাজন দেখুন
এসআর প্রতিবেদন (SR Report)

এসআরভিত্তিক বিক্রয় প্রতিবেদন (SR Wise Sales Report)

প্রতিটি সেলস রিপ্রেজেন্টেটিভ (SR) এর বিক্রয় কর্মক্ষমতা দেখুন। কোন SR সবচেয়ে বেশি বিক্রি করছে, তাদের মোট বিক্রয় পরিমাণ ও কমিশন হিসাব।

  1. MIS Report → SR Report → SR Wise Sales Report ক্লিক করুন
  2. তারিখ সীমা নির্ধারণ করুন
  3. প্রতিটি SR এর বিক্রয় পরিমাণ তুলনামূলক দেখুন
  4. সর্বোচ্চ বিক্রেতা SR শনাক্ত করুন

এসআরভিত্তিক বিক্রয় বিবরণ (SR Wise Sales Details)

নির্দিষ্ট SR এর বিক্রয়ের বিস্তারিত বিবরণ - কোন গ্রাহককে কোন প্রোডাক্ট বিক্রি করেছে তার সম্পূর্ণ তালিকা।

  1. MIS Report → SR Report → SR Wise Sales Details ক্লিক করুন
  2. SR নির্বাচন করুন ড্রপডাউন থেকে
  3. তারিখ সীমা দিন
  4. সকল বিক্রয়ের বিস্তারিত তালিকা দেখুন

এসআরভিত্তিক গ্রাহক বিক্রয় সারসংক্ষেপ (SR Wise Customer Sales Summary)

এসআর ভিজিট প্রতিবেদন (SR Visit Report)

এসআরভিত্তিক গ্রাহক অবস্থা (SR Wise Customer Status)

এসআরভিত্তিক নগদ আদায় (SR Wise Cash Collection)

এসআর কমিশন প্রতিবেদন (SR Commission Report)

নির্দিষ্ট তারিখ পর্যন্ত সকল গ্রাহকের বকেয়া পরিমাণ। যেকোনো তারিখে মোট গ্রাহক বকেয়া জানতে চাইলে এই রিপোর্ট ব্যবহার করুন।

ব্যয় প্রতিবেদন (Expense Report)

সকল খরচ বা ব্যয়ের বিস্তারিত প্রতিবেদন। কোন খাতে কত খরচ হচ্ছে তা বিশ্লেষণ করুন। খরচ নিয়ন্ত্রণ ও বাজেট পরিকল্পনায় সাহায্য করে।

  1. MIS Report → Management Report → Expense Report ক্লিক করুন
  2. তারিখ সীমা ও খাত নির্বাচন করুন
  3. খাতভিত্তিক ব্যয়ের তালিকা দেখুন
  4. মোট ব্যয়ের সামগ্রিক হিসাব দেখুন

প্রোডাক্টভিত্তিক লাভ প্রতিবেদন (Product Wise Benefit Report)

কোন প্রোডাক্টে কত লাভ হচ্ছে তার বিস্তারিত প্রতিবেদন। প্রতিটি প্রোডাক্টের ক্রয়মূল্য ও বিক্রয়মূল্যের পার্থক্য দেখুন। সবচেয়ে লাভজনক প্রোডাক্ট চিহ্নিত করে ব্যবসার কৌশল নির্ধারণে সাহায্য করে।

  1. MIS Report → Management Report → Product Wise Benefit Report ক্লিক করুন
  2. ক্যাটাগরি বা ব্র্যান্ড ফিল্টার দিন
  3. প্রতিটি প্রোডাক্টের ক্রয়মূল্য, বিক্রয়মূল্য ও লাভ দেখুন
  4. লাভের শতকরা হার অনুযায়ী সাজান
এই রিপোর্ট থেকে বুঝতে পারবেন কোন প্রোডাক্টে বেশি লাভ এবং কোনটিতে কম। কম লাভের প্রোডাক্টের দাম বাড়ানো বা বেশি লাভের প্রোডাক্টের স্টক বাড়ানোর সিদ্ধান্ত নিন।

কনফিগারেশন (Configuration)

কোম্পানি সেটিংস (Company Settings)

আপনার প্রতিষ্ঠানের সাধারণ তথ্য ও সেটিংস কনফিগার করুন। কোম্পানির নাম, ঠিকানা, লোগো, ফোন নম্বর, ইমেইল ইত্যাদি তথ্য এখানে সেট করা হয়। এই তথ্য চালান, রিপোর্ট ও পিডিএফের হেডারে ব্যবহৃত হয়।

  1. সাইডবার থেকে System Settings → Company Settings ক্লিক করুন
  2. কোম্পানির নাম, ঠিকানা, ফোন, ইমেইল লিখুন
  3. লোগো আপলোড করুন (ঐচ্ছিক)
  4. অন্যান্য সেটিংস পরিবর্তন করুন
  5. Save ক্লিক করুন
কোম্পানি সেটিংস সবচেয়ে প্রথমে সেটআপ করুন। এই তথ্য ছাড়া চালান ও রিপোর্টে কোম্পানির নাম দেখাবে না।

চালান টেমপ্লেট (Invoice Templates)

বিক্রয় চালান বা ইনভয়েসের ডিজাইন কাস্টমাইজ করুন। কোম্পানির লোগো, ঠিকানা, শর্তাবলী, ফন্ট সাইজ ইত্যাদি পরিবর্তন করতে পারবেন। বিভিন্ন ধরনের চালানের জন্য আলাদা টেমপ্লেট তৈরি করা যায়।

  1. System Settings → Invoice Templates ক্লিক করুন
  2. বিদ্যমান টেমপ্লেট সম্পাদনা করুন বা নতুন তৈরি করুন
  3. হেডার, ফুটার, বডি সেকশন কাস্টমাইজ করুন
  4. লোগো ও কোম্পানি তথ্য যোগ করুন
  5. প্রিভিউ দেখুন ও সেভ করুন

নম্বর ফরম্যাট (Number Formats)

সিস্টেমের সকল নম্বরিং ফরম্যাট কনফিগার করুন। চালান নম্বর, ক্রয় অর্ডার নম্বর, সেলস অর্ডার নম্বর ইত্যাদির ফরম্যাট ও প্রিফিক্স নির্ধারণ করুন। যেমন: INV-00001, PUR-00001, SAL-00001 ইত্যাদি।

  1. System Settings → Number Formats ক্লিক করুন
  2. প্রতিটি ডকুমেন্ট টাইপের জন্য প্রিফিক্স সেট করুন
  3. নম্বরের দৈর্ঘ্য নির্ধারণ করুন (যেমন: ৫ ডিজিট)
  4. বর্তমান কাউন্টার ভ্যালু দেখুন
  5. Save ক্লিক করুন
নম্বর ফরম্যাট একবার সেট করলে প্রিফিক্স পরিবর্তন করবেন না। প্রিফিক্স পরিবর্তন করলে আগের ডকুমেন্টের নম্বর সিরিজের সাথে নতুনগুলো মিলবে না।
অডিট ও অনুসন্ধান (Audit & Search)

অডিট ট্রেইল ভিউয়ার (Audit Trail Viewer)

সিস্টেমের সকল কার্যক্রমের লগ বা অডিট ট্রেইল দেখুন। কে কোন পেজে কখন কী পরিবর্তন করেছে তার সম্পূর্ণ রেকর্ড। ডেটা সততা যাচাই ও নিরাপত্তা পর্যবেক্ষণের জন্য অপরিহার্য।

  1. System Settings → Audit Trail Viewer ক্লিক করুন
  2. তারিখ, ব্যবহারকারী বা মডিউল অনুযায়ী ফিল্টার করুন
  3. প্রতিটি কার্যক্রমের বিস্তারিত দেখুন - ব্যবহারকারী, সময়, পরিবর্তন
  4. PDF/CSV তে রপ্তানি করুন
অডিট ট্রেইল সিস্টেম দ্বারা স্বয়ংক্রিয়ভাবে রেকর্ড হয়। এটি ম্যানুয়ালি সম্পাদনা বা মুছে ফেলা যায় না।

পারফরম্যান্স ও ক্যাশ (Performance & Cache)

সিস্টেমের পারফরম্যান্স পর্যবেক্ষণ ও ক্যাশ ব্যবস্থাপনা। ক্যাশ ক্লিয়ার করে সিস্টেমের গতি বাড়ানো যায়। মেমরি ব্যবহার, ডেটাবেস কানেকশন ও সিস্টেম রিসোর্স মনিটর করুন।

  1. System Settings → Performance & Cache ক্লিক করুন
  2. সিস্টেমের বর্তমান পারফরম্যান্স মেট্রিক্স দেখুন
  3. ক্যাশ ক্লিয়ার করতে Clear Cache বাটনে ক্লিক করুন
  4. সিস্টেম স্বাস্থ্য সূচক দেখুন
সিস্টেম ধীর হলে ক্যাশ ক্লিয়ার করুন। তবে মনে রাখবেন, ক্যাশ ক্লিয়ার করলে কিছুক্ষণ সিস্টেম আরও ধীর হতে পারে কারণ ক্যাশ পুনরায় তৈরি হতে সময় লাগে।

সম্পূর্ণ ব্যবহার প্রবাহ (Complete Usage Workflow)

এই অংশে আপনার দৈনন্দিন ব্যবসার সবচেয়ে গুরুত্বপূর্ণ কাজগুলো কীভাবে সম্পন্ন করবেন তার ধাপে ধাপে নির্দেশনা দেওয়া হলো।

১. কীভাবে ব্র্যান্ড কোম্পানি থেকে প্রোডাক্ট কিনবেন

  1. কোম্পানি নিবন্ধন: Core Config → Companies পেজে ব্র্যান্ড কোম্পানির তথ্য যোগ করুন (যেমন: Samsung, Sony)
  2. সরবরাহকারী নিবন্ধন: Customers & Suppliers → Suppliers পেজে ডিলার/সাপ্লায়ার যোগ করুন
  3. প্রোডাক্ট সেটআপ: Core Config → Products পেজে প্রোডাক্টের নাম, মডেল, ক্রয়মূল্য, বিক্রয়মূল্য ও গোডাউন নির্ধারণ করুন
  4. অর্ডার শিট: Order Sheet → Company Ordersheet পেজে কোম্পানিকে অর্ডার পাঠান
  5. ক্রয় অর্ডার: Purchase Order পেজে আনুষ্ঠানিক PO তৈরি করুন - সরবরাহকারী, প্রোডাক্ট ও পরিমাণ নির্বাচন করুন
  6. প্রোডাক্ট রিসিভ: PO তে "Receive" বাটনে ক্লিক করে প্রোডাক্ট গোডাউনে গ্রহণ করুন - স্টক স্বয়ংক্রিয়ভাবে বাড়বে
  7. পেমেন্ট: Cash Delivery বা Bank Transaction পেজে সরবরাহকারীকে পেমেন্ট রেকর্ড করুন

২. কীভাবে বিভিন্ন গোডাউনে প্রোডাক্ট এড করবেন

  1. গোডাউন সেটআপ: Structure → Godowns পেজে মেইন গোডাউন, শোরুম ও শাখা গোডাউন তৈরি করুন
  2. ধারণক্ষমতা: Structure → Capacity পেজে প্রতিটি গোডাউনের ধারণক্ষমতা নির্ধারণ করুন
  3. প্রোডাক্ট গোডাউন নির্ধারণ: Products পেজে প্রতিটি প্রোডাক্টের ডিফল্ট গোডাউন সেট করুন
  4. স্থানান্তর: Stock → Transfer পেজে এক গোডাউন থেকে অন্য গোডাউনে প্রোডাক্ট ট্রান্সফার করুন
  5. স্টক দেখুন: Stock পেজে প্রতিটি গোডাউনের বর্তমান স্টক দেখুন

৩. কীভাবে ডিলারের প্রোডাক্ট বিক্রয় করবেন

  1. গ্রাহক নিবন্ধন: Customers পেজে গ্রাহকের তথ্য যোগ করুন (নাম, ফোন, ক্রেডিট লিমিট)
  2. সেলস অর্ডার: Sales Order পেজে নতুন অর্ডার তৈরি করুন - গ্রাহক, প্রোডাক্ট ও পরিমাণ নির্বাচন করুন
  3. পেমেন্ট গ্রহণ: নগদ, কার্ড বা ক্রেডিট পেমেন্ট অপশন নির্বাচন করুন
  4. চালান তৈরি: সেলস অর্ডার সেভ করলে স্বয়ংক্রিয়ভাবে চালান/ইনভয়েস তৈরি হবে
  5. স্টক আপডেট: বিক্রয়ের পর স্টক স্বয়ংক্রিয়ভাবে কমবে

৪. কীভাবে কিস্তিতে প্রোডাক্ট বিক্রয় করবেন

  1. সুদের হার সেটআপ: Structure → Interest % Engine পেজে কিস্তির সুদের হার কনফিগার করুন
  2. কিস্তি বিক্রয়: Hire Sales পেজে নতুন কিস্তি বিক্রয় তৈরি করুন - গ্রাহক, প্রোডাক্ট, ডাউন পেমেন্ট, কিস্তির সংখ্যা ও সুদের হার নির্ধারণ করুন
  3. EMI হিসাব: সিস্টেম স্বয়ংক্রিয়ভাবে মাসিক কিস্তির পরিমাণ (EMI) হিসাব করবে
  4. কিস্তি আদায়: প্রতি মাসে MIS Report → Hire Sales Report → Installment Collection পেজে কিস্তি আদায় রেকর্ড করুন
  5. খেলাপি পর্যবেক্ষণ: Defaulting Customer রিপোর্ট দেখে কিস্তি আদায় নিশ্চিত করুন

৫. কীভাবে নগদে প্রোডাক্ট বিক্রয় করবেন

  1. সেলস অর্ডার: Sales Order পেজে নতুন অর্ডার তৈরি করুন
  2. গ্রাহক নির্বাচন: গ্রাহক নির্বাচন করুন (ওয়াক-ইন গ্রাহকও তৈরি করতে পারেন)
  3. পেমেন্ট পদ্ধতি: "Cash" পেমেন্ট অপশন নির্বাচন করুন
  4. চালান প্রিন্ট: সেলস অর্ডার সেভ করলে ইনভয়েস তৈরি হবে - PDF ডাউনলোড করে প্রিন্ট করুন
  5. নগদ জমা: Cash Collection পেজে নগদ অর্থ জমার রেকর্ড করুন

৬. কীভাবে লাভ-ক্ষতির হিসাব দেখবেন

  1. লাভ-ক্ষতি হিসাব: Accounting Report → Profit and Loss Account পেজে যান
  2. তারিখ সীমা: যে সময়কালের হিসাব দেখতে চান তারিখ সীমা নির্ধারণ করুন
  3. আয়ের হিসাব: মোট বিক্রয় আয় ও অন্যান্য আয় দেখুন
  4. ব্যয়ের হিসাব: মোট ক্রয় খরচ, পরিচালন ব্যয় ও অন্যান্য খরচ দেখুন
  5. নিট লাভ/ক্ষতি: মোট আয় - মোট ব্যয় = নিট লাভ বা ক্ষতি
  6. উদ্বৃত্তপত্র: Balance Sheet দেখে সম্পত্তি ও দায়ের সামগ্রিক চিত্র দেখুন
লাভ-ক্ষতির হিসাব সঠিক দেখতে সব আয়-ব্যয়ের এন্ট্রি নিয়মিত করতে হবে। কোনো খরচ বাদ পড়লে লাভ বেশি দেখাবে।

৭. কীভাবে লো স্টক দেখবেন ও প্রোডাক্ট কিনবেন

  1. স্টক পূর্বাভাস: MIS Report → Basic Report → Stock Forcasting (Product Wise) পেজে যান
  2. লো স্টক প্রোডাক্ট: যেসব প্রোডাক্টের স্টক সীমার নিচে আছে তাদের তালিকা দেখুন
  3. স্বয়ংক্রিয় PO: Inventory → Auto PO পেজে গিয়ে লো স্টক প্রোডাক্টের জন্য স্বয়ংক্রিয়ভাবে ক্রয় অর্ডার তৈরি করুন
  4. অথবা ম্যানুয়াল PO: Purchase Order পেজে নিজে ক্রয় অর্ডার তৈরি করুন
  5. স্টক ট্রেন্ড: Stock Trend Analysis দেখে প্রোডাক্টের চাহিদা প্রবণতা বুঝুন
লো স্টক অ্যালার্ট পেতে Stock Forcasting রিপোর্ট নিয়মিত দেখুন। Auto PO ব্যবহার করলে স্বয়ংক্রিয়ভাবে প্রোডাক্ট কেনার প্রক্রিয়া শুরু হবে।

৮. কীভাবে বেশি বিক্রি হওয়া প্রোডাক্ট চিনবেন

  1. বিক্রয় কর্মক্ষমতা: MIS Report → Basic Report → Sales Performance পেজে যান
  2. মডেলভিত্তিক বিক্রয়: MIS Report → Sales Report → Model Wise Sales পেজে কোন মডেল সবচেয়ে বেশি বিক্রি হচ্ছে দেখুন
  3. প্রোডাক্টভিত্তিক লাভ: MIS Report → Management Report → Product Wise Benefit Report পেজে কোন প্রোডাক্টে সবচেয়ে বেশি লাভ দেখুন
  4. স্টক ট্রেন্ড: Stock Trend Analysis দেখে বিক্রয় প্রবণতা বুঝুন
  5. কৌশল নির্ধারণ: বেশি বিক্রির প্রোডাক্টের স্টক বাড়ান ও কম বিক্রির প্রোডাক্ট কম কিনুন
